A full-stack web application built using Django that allows patients to book doctor appointments easily and enables admins to manage them efficiently.

## 🚀 Live Demo

🔗https://clinic-project-zqsm.onrender.com/

## ✨ Features

* 📅 Book doctor appointments
* 👨‍⚕️ View available doctors
* 📊 Admin dashboard
* ✅ Approve / ❌ Reject appointments
* 🗑 Delete appointments
* 🎨 Attractive UI design
* 🔐 Admin login system

## 🛠 Tech Stack

* Python 🐍
* Django 🌐
* HTML5 & CSS3 🎨
* SQLite 🗄
* Render (Deployment) 🚀
